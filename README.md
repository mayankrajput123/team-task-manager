# Team Task Manager 🚀

A full-stack MERN application to manage projects and tasks with authentication.

## 🔧 Tech Stack

Frontend:
- React.js
- Bootstrap
- Axios

Backend:
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

Deployment:
- Railway
- GitHub

---

## ✨ Features

- User Registration & Login
- JWT Authentication
- Create Projects
- Create Tasks
- Due Date Support
- Overdue Detection
- Task Status Update
- Dashboard Stats
- Protected Routes
